# gonav

源项目 https://github.com/dengxiwang/go-nav ，action构建并生成方便linux_64平台直接运行运行的文件，使用[bun](https://github.com/oven-sh/bun)作为启动器

# 目录
--目录
----.next
----data #数据目录
----node_modules
----public #文件目录
----.env.local #环境变量
----package.json
----server.js
----bun #运行器

# 启动命令
```
./bun --smol server.js
```
